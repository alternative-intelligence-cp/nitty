# Nitty Quick Reference Card

## Tab Management
| Shortcut | Action |
|---|---|
| **Ctrl+Shift+T** | New Tab |
| **Ctrl+Shift+W** | Close Tab |
| **Ctrl+Tab** | Next Tab |
| **Ctrl+Shift+Tab** | Previous Tab |
| **Alt+1** through **9** | Switch to Tab 1-9 |

## Pane Management
| Shortcut | Action |
|---|---|
| **Ctrl+Shift+H** | Split Horizontally |
| **Ctrl+Shift+V** | Split Vertically |
| **Ctrl+Shift+X** | Close Pane |
| **Alt+Arrow Keys** | Navigate Panes |

## Edit & View
| Shortcut | Action |
|---|---|
| **Ctrl+Shift+C** | Copy |
| **Ctrl+Shift+V** | Paste |
| **Ctrl+Shift+F** | Search Scrollback |
| **Ctrl+Shift+K** | Clear Screen & Buffer |
| **Ctrl+Shift+=** | Zoom In |
| **Ctrl+Shift+-** | Zoom Out |
| **Ctrl+Shift+0** | Reset Zoom |
| **F11** | Fullscreen |
| **F10** | Maximize |

## Application
| Shortcut | Action |
|---|---|
| **Ctrl+Shift+B** | Connection Manager (SSH/Serial) |
| **Ctrl+Shift+S** | SFTP File Browser |
| **Ctrl+Shift+P** | Plugin Manager |
| **Ctrl+Shift+,** | Settings |
| **F12** | Toggle Quake Mode (if enabled) |

## Specialized & Serial
| Shortcut | Action |
|---|---|
| **F8** | Toggle Hexdump Mode (Serial) |
| **Ctrl+Shift+\`** | Send BREAK Signal (Serial) |
| **Ctrl+F6** | Skip Navigation (Screen Reader) |

## Quick Commands
- Type `user@host` in the Connection Manager to quickly start an SSH session.
- Run `sz filename` on a remote serial device to trigger an automatic Zmodem download.
